使い方
1. 取り敢えず一番下にあるコメントアウトされている関数で
directory以下の(require :)を作ります。ただし、文字列だったり、
#:keywordに対応していませんので、手作業で除くか、関数を改良します。

2. requireを沢山実行します。

3. その後、以下の関数を実行します。
get-lisp-symbols  とあるファイル(べた書き)に関数名 パッケージ 引数 型 結果の型 の一覧を作る
get-lisp-symbols2 とあるファイル(べた書き)に頻出関数トップ8割を出します。
get-lisp-symbols3 標準出力にファイル毎の頻出関数を出します。膨大で意味がないです。改良してパッケージ毎がいいかも。
